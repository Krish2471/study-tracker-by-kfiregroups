# 🚀 HASH — Premium Study Tracker by KFIRE GROUPS

![HASH Banner](https://raw.githubusercontent.com/Krish2471/study-tracker-by-kfiregroups/main/public/logo.png)

> **Study Smarter, Level Up.** 

HASH is a state-of-the-art, gamified study ecosystem designed to turn your academic journey into an immersive adventure. Built with cutting-edge web technologies, HASH provides a seamless, high-performance interface for modern students.

---

## ✨ Key Features

- 🕒 **Deep Focus Timer**: Distraction-free Pomodoro-style timer with ambient soundscapes.
- 📓 **Scribble Notebook**: A beautiful, fluid canvas for digital note-taking and sketching.
- 📅 **Smart Planner**: Integrated calendar to sync exams, goals, and daily tasks.
- 🃏 **SRS Flashcards**: Spaced Repetition System to master subjects faster.
- 🎮 **Gamified Growth**: Earn coins, track streaks, and level up your custom avatar.
- 📊 **Advanced Analytics**: Detailed insights into your focus hours and task completion.
- 🤖 **AI Study Assistant**: (Coming Soon) Personal AI to help summarize notes and generate quizzes.

## 🛠️ Technology Stack

- **Core**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: [Zustand](https://docs.pmnd.rs/zustand/)
- **Database/Auth**: [Firebase](https://firebase.google.com/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Krish2471/study-tracker-by-kfiregroups.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
```bash
npm run build
```
